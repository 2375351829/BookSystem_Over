# 城市图书馆管理系统 - 用户操作手册

## 1. 手册概述

### 1.1 系统简介
城市图书馆管理系统是一套现代化的图书馆信息管理平台，旨在为读者提供便捷的图书借阅、预约、座位管理等服务，同时为图书馆管理员提供高效的图书管理、用户管理和数据统计功能。

本系统采用前后端分离架构，前端基于Vue 3 + Element Plus构建，提供友好的用户界面；后端基于Spring Boot框架，确保系统的稳定性和安全性。系统支持中英文双语切换，满足不同用户的需求。

**系统主要功能包括：**
- 图书检索与浏览：支持关键词搜索、分类筛选、高级搜索等
- 图书借阅管理：在线借书、续借、归还等操作
- 图书预约：预约已借出的图书，到货后取阅
- 座位预约：在线选择阅览室座位，支持签到签退
- 参考咨询：向图书馆提交咨询问题并查看回复
- 消息通知：接收借阅、归还、到期提醒等各类通知
- 个人中心：管理个人信息、查看历史记录、收藏图书等
- 管理后台：图书管理、用户管理、统计分析等功能

### 1.2 适用读者
本手册适用于以下用户群体：

| 用户类型 | 说明 | 主要权限 |
|---------|------|---------|
| **普通读者（用户）** | 在图书馆注册的普通读者 | 借阅图书、预约图书和座位、提交咨询、查看个人记录 |
| **图书管理员** | 负责图书馆日常运营的工作人员 | 管理图书、处理借还、回复咨询、查看统计数据 |
| **系统管理员** | 负责系统维护和技术支持的高级管理人员 | 系统配置、权限管理、日志查看、数据导出 |

### 1.3 使用前准备

#### 系统访问地址
- **本地开发环境**：`http://localhost:5173`（前端）+ `http://localhost:8080`（后端API）
- **生产环境**：请联系系统管理员获取具体的访问地址

#### 浏览器要求
为确保最佳使用体验，推荐使用以下浏览器：
- ✅ **推荐浏览器**：Google Chrome 90+、Microsoft Edge 90+、Firefox 88+
- ⚠️ **兼容浏览器**：Safari 14+（部分功能可能存在兼容性问题）
- ❌ **不支持的浏览器**：Internet Explorer（已停止维护）

**浏览器设置要求：**
- 启用JavaScript（系统必须依赖JavaScript运行）
- 启用Cookie（用于保持登录状态）
- 允许弹出窗口（部分确认对话框需要）

#### 账号获取方式
1. **新用户注册**：首次使用需通过注册页面创建账号（详见第2.2节）
2. **管理员分配**：部分账号由系统管理员统一创建和分配
3. **忘记密码**：如忘记密码，可通过登录页面的"忘记密码"链接找回，或联系管理员重置

## 2. 快速入门指南

### 2.1 首次访问系统

**步骤1：打开浏览器**
在浏览器地址栏输入系统访问地址，按回车键进入系统。

**步骤2：进入登录页面**
系统会自动跳转到登录页面（LoginView），页面布局如下：
- **左侧区域**：展示系统欢迎信息和主要功能介绍（图书资源、智能检索、便捷服务等）
- **右侧区域**：登录表单，包含用户名、密码、验证码输入框

**步骤3：语言切换**
登录页面右上角提供语言切换按钮，可在中文（简体）和英文之间切换。

### 2.2 注册新账号

如果您是第一次使用本系统，需要先注册一个账号。

#### 注册步骤详解

**步骤1：找到注册入口**
在登录页面底部或相关位置找到"注册"链接/按钮，点击进入注册页面。

**步骤2：填写注册信息**
注册表单通常包含以下必填字段：

| 字段名称 | 说明 | 填写要求 | 示例 |
|---------|------|---------|------|
| **用户名** | 登录时使用的唯一标识 | 3-20个字符，支持字母、数字、下划线 | zhangsan2024 |
| **密码** | 账号安全凭证 | 6-20个字符，建议包含字母和数字 | Abc@123456 |
| **确认密码** | 再次输入密码以确认 | 必须与密码完全一致 | Abc@123456 |
| **姓名** | 您的真实姓名 | 2-20个中文字符 | 张三 |
| **邮箱** | 用于接收通知和找回密码 | 有效的邮箱格式 | zhangsan@example.com |
| **手机号** | 用于接收短信通知（可选） | 11位手机号码 | 13800138000 |
| **验证码** | 图形验证码 | 输入图片显示的字符 | （见图片） |

**步骤3：阅读并同意服务条款**
勾选"我已阅读并同意《用户服务协议》和《隐私政策》"复选框。

**步骤4：点击注册**
填写完所有必填信息后，点击"注册"按钮完成注册。

#### 注意事项
⚠️ **重要提示：**
- 用户名一旦注册成功不可修改，请谨慎选择
- 密码请妥善保管，不要与他人分享
- 建议使用强密码（包含大小写字母、数字和特殊符号）
- 邮箱和手机号用于账号安全和通知推送，请确保真实有效
- 同一邮箱或手机号只能注册一个账号

### 2.3 登录系统

#### 登录步骤

**步骤1：访问登录页面**
打开系统首页，如果未登录会自动跳转到登录页面。

**步骤2：输入登录凭据**

1. **用户名**：在"用户名"输入框中输入您注册时的用户名
   - 支持用户名、邮箱或手机号登录（具体取决于系统配置）
   
2. **密码**：在"密码"输入框中输入您的密码
   - 密码输入时会显示为圆点（•）以保护隐私
   - 可点击眼睛图标👁️显示/隐藏密码
   
3. **验证码**：在"验证码"输入框输入图片显示的字符
   - 验证码不区分大小写
   - 如果看不清楚，可点击图片刷新验证码
   - 验证码有效期通常为5分钟

**步骤3：可选操作**
- ☑️ **记住我**：勾选后下次访问无需再次输入密码（7天内有效）
- 🔗 **忘记密码**：点击链接可进行密码找回流程

**步骤4：点击登录**
点击蓝色的"登录"按钮，系统会验证您的身份信息。

**步骤5：进入系统**
登录成功后会自动跳转到首页（HomeView），您可以开始使用各项功能。

#### 忘记密码处理

如果您忘记了密码，可以按照以下步骤重置：

1. **在登录页面点击"忘记密码"链接**
2. **输入注册时使用的邮箱或手机号**
3. **系统会发送重置链接/验证码到您的邮箱/手机**
4. **点击链接或输入验证码进入重置页面**
5. **设置新密码并确认**
6. **使用新密码重新登录**

#### 常见登录问题

| 问题现象 | 可能原因 | 解决方法 |
|---------|---------|---------|
| 提示"用户名或密码错误" | 密码输入错误 | 点击密码框的眼睛图标确认输入，注意大小写 |
| 提示"验证码错误" | 验证码过期或输入错误 | 点击验证码图片刷新后重新输入 |
| 提示"账号已被禁用" | 违反规定被禁用 | 联系管理员了解原因并申请解封 |
| 页面无法加载 | 网络连接问题 | 检查网络连接，刷新页面重试 |
| 提示"账号不存在" | 用户名输入错误 | 确认用户名是否正确，区分大小写 |

## 3. 读者操作指南

本章详细介绍普通读者（用户）可使用的所有功能和操作方法。

### 3.1 图书检索与浏览

图书检索是系统的核心功能之一，帮助您快速找到所需的图书资源。

#### 如何搜索图书

**方式一：快速搜索（首页）**
1. 进入系统首页（HomeView）
2. 在页面顶部的搜索框中输入关键词（书名、作者、ISBN等）
3. 按**回车键**或点击搜索按钮🔍
4. 系统将显示匹配的图书列表

**方式二：高级搜索（图书列表页）**
1. 点击导航菜单的"图书浏览"或"图书列表"
2. 进入BookListView页面，使用筛选条件：
   - 关键词搜索（书名/作者/ISBN）
   - 图书分类（A-F、G-H、I-J等中图法分类）
   - 图书状态（在馆可借/已借出/已预约）
   - 语言（中文/英文/日文/其他）
   - 排序方式（默认/最新上架/评分最高/借阅最多）

**高级搜索选项：**
- ISBN精确匹配
- 出版社筛选
- 出版年份范围
- 价格范围

#### 视图模式切换
搜索结果支持两种显示模式：
- **网格视图（默认）**：以卡片形式展示图书封面和信息
- **列表视图**：以表格形式展示详细信息

#### 热门图书推荐
首页展示热门图书轮播区和新书上架区域，方便您发现优质图书。

### 3.2 图书借阅操作

#### 借书步骤详解

**前提条件：**
- ✅ 已成功登录系统
- ✅ 账号状态正常（未被禁用）
- ✅ 未达到借阅上限数量
- ✅ 目标图书状态为"在馆可借"

**操作步骤：**
1. 搜索并找到想要借阅的图书
2. 点击进入图书详情页（BookDetailView）
3. 查看图书信息（书名、作者、ISBN、馆藏位置等）
4. 点击"借阅此书"按钮
5. 在确认对话框中点击"确认借阅"
6. 系统显示借阅成功消息和应还日期

#### 借阅规则说明

| 规则项目 | 普通读者 | VIP读者/教师 |
|---------|---------|-------------|
| **最大借阅数量** | 5本 | 10本 |
| **单次借阅期限** | 30天 | 60天 |
| **最大续借次数** | 2次 | 3次 |
| **每次续借天数** | 15天 | 30天 |
| **逾期罚款费率** | 0.1元/天/本 | 0.05元/天/本 |

#### 如何续借

**续借条件：**
- ✅ 图书尚未归还且未逾期
- ✅ 续借次数未达上限
- ✅ 图书未被他人预约

**续借步骤：**
1. 进入"借阅管理"页面（Borrow.vue）
2. 在"已借阅"标签页找到目标图书
3. 点击"续借"按钮（🔄图标）
4. 系统显示续借成功和新到期日期

> ⚠️ 注意：逾期图书不可续借，已达续借上限的图书续借按钮呈灰色。

#### 归还图书

**线上归还：**
1. 进入"借阅管理"页面
2. 在"已借阅"标签页找到要归还的图书
3. 点击"归还"按钮（↩️图标）

**线下归还：**
- 携带图书到图书馆服务台
- 由工作人员扫描办理

**逾期处理：**
- 逾期会产生罚款（逾期天数 × 日罚款率 × 图书数量）
- 请尽量按时归还

### 3.3 图书预约

#### 何时需要预约
- 目标图书所有副本都已借出
- 希望该书归还后优先为您保留

#### 预约操作步骤
1. 搜索目标图书，确认状态为"已借出"
2. 点击"预约此书"按钮
3. 确认预约信息
4. 预约成功后可在"预约管理"页面查看状态

#### 预约状态说明

| 状态 | 含义 | 可执行操作 |
|-----|------|-----------|
| 预约中(PENDING) | 正在排队等待 | 取消预约 |
| 待取书(READY) | 已到馆，请领取 | 取消预约 |
| 已完成(COMPLETED) | 成功领取 | 无 |
| 已取消(CANCELLED) | 已取消 | 无 |
| 已过期(EXPIRED) | 超时未取 | 无 |

#### 取消预约
- 仅"预约中"和"待取书"状态可取消
- 在预约列表中点击"取消预约"按钮即可

#### 预约有效期
- 排队等待期：7-14天
- 到馆取书期：3天内必须领取
- 过期需重新预约

---

### 3.4 座位预约

座位预约让您可以提前预订阅览室的学习座位。

#### 操作流程

**步骤1-2：选择阅览室和日期**
- 从下拉菜单选择阅览室（查看位置和设施）
- 选择今天或未来7天内的日期

**步骤3-4：设置时间段并选择座位**
- 选择开始和结束时间（08:00-21:30，每30分钟一档）
- 在座位布局图中点击绿色（可用）座位
- 查看座位特性（插座🔌、靠窗☀️）

**步骤5-6：确认预约**
- 点击"确认预约"按钮
- 核对信息后确认完成

#### 签到签退

**签到：**
- 预约开始前15分钟至开始后30分钟内可签到
- 在"我的预约"列表中点击"签到"按钮

**签退：**
- 离开时点击"签退"按钮
- 建议主动签退以释放座位资源

#### 违规注意事项

**违规行为及后果：**

| 行为 | 后果 |
|-----|------|
| 迟到超过30分钟未签到 | 记违规1次 |
| 无故爽约 | 记违规1次 |
| 累计3次违规 | 禁止预约座位7天 |

**避免违规的建议：**
1. 准时到达，预留找座时间
2. 设置提醒，避免遗忘
3. 离开时记得签退
4. 如不能前往，提前30分钟取消

### 3.5 个人中心

个人中心（Profile.vue）用于管理账户信息。

#### 可执行操作
- **修改个人信息**：修改姓名等基本信息
- **修改密码**：定期更换密码保障安全
- **查看借阅记录**：已借阅/已归还列表，支持续借和归还
- **查看预约记录**：图书预约状态和管理
- **查看收藏图书**：管理和浏览收藏的图书
- **查看罚款记录**：查询和缴纳罚款
- **查看历史记录**：借阅/预约/座位/咨询历史

### 3.6 参考咨询

通过咨询中心（Inquiry.vue）向图书馆提问。

#### 提交咨询
1. 点击"提交咨询"按钮
2. 填写标题（5-100字）、分类（借阅/账户/推荐/其他）、内容（10-500字）
3. 提交后状态为"待处理"

#### 查看回复
- 咨询列表显示待处理/已回复/已关闭数量
- 点击"查看详情"可看到管理员的回复内容
- 回复后系统会发送通知

---

### 3.7 消息通知

通知中心（Notification.vue）汇总各类系统通知。

#### 通知类型
- **BORROW**：借阅通知（蓝色）
- **RETURN**：归还通知（绿色）
- **DUE_REMINDER**：到期提醒（黄色）
- **OVERDUE**：逾期通知（红色）
- **SYSTEM**：系统公告（灰色）
- **RESERVATION**：预约通知（橙色）
- **SEAT**：座位通知（紫色）

#### 操作功能
- 单条标记已读/批量全部标记已读
- 按类型和阅读状态筛选
- 通知保留30-90天

---

### 3.8 图书评价与收藏

#### 发表书评
- 在图书详情页的评价区域撰写
- 评分（1-5星）+ 标题 + 正文（50-1000字）
- 内容需真实、文明、具体

#### 收藏图书
- 点击图书卡片的❤️图标收藏
- 单用户最大收藏约500-1000本
- 在"我的收藏"页面统一管理

---

### 3.9 数据导出（如有权限）

支持导出个人数据：
- 借阅记录、预约记录、座位使用记录
- 收藏列表、综合报告
- 格式：Excel/CSV/PDF
- 导出文件保留7天

## 4. 图书管理员操作指南

### 4.1 管理员仪表盘（AdminDashboardView.vue）

**数据概览卡片：**
- 总藏书量、今日借阅/归还、逾期未还、活跃用户数

**图表分析：**
- 借阅趋势折线图
- 分类分布饼图
- 热门图书Top10柱状图
- 待处理事项时间线
- 用户活跃度排行

**快捷操作入口：**
- 新增图书、批量导入、用户管理、查看报表

### 4.2 图书管理（AdminBookView.vue）

#### 新增图书（AdminBookAddView.vue）
填写ISBN、书名、作者、出版社、分类、复本数等信息，上传封面后保存。

#### 编辑/删除图书
- 编辑：修改图书各项信息
- 删除：需确认无未归还借阅和无活跃预约（二次确认）

#### 批量导入（AdminBookImportView.vue）
1. 下载Excel模板
2. 按格式填写数据（ISBN必填且唯一）
3. 上传文件并预览校验
4. 执行导入（支持最多约1000条/次）

#### 图书上下架
- 上架：恢复为"在馆"可借状态
- 下架：隐藏图书（破损/遗失/审查等原因）

### 4.3 用户管理（AdminUserView.vue）

- 查看/编辑用户信息
- 禁用/启用账号（选择原因）
- 重置密码（生成随机密码或指定密码）

### 4.4 借阅管理（AdminBorrowView.vue）

#### 手动借阅
扫描或输入读者/图书信息，系统自动检查规则后办理。

#### 办理归还
扫码或查找借阅记录，确认信息后办理，自动计算逾期罚款。

#### 处理逾期
- 查看逾期列表
- 发送催还通知

## 6. 常见问题解答（FAQ）

### 6.1 账号相关问题

**Q: 无法登录怎么办？**
A: 检查用户名密码（区分大小写）、验证码是否正确；确认账号未被禁用；检查网络；清除缓存；联系管理员。

**Q: 忘记密码如何处理？**
A: 使用"忘记密码"自助找回，或联系管理员重置（获取临时密码后立即修改）。

**Q: 账号被禁用怎么办？**
A: 了解原因（逾期/违约/安全风险），解决问题后申请解封。申诉流程通常1-3个工作日。

### 6.2 借阅相关问题

**Q: 为什么无法借阅？**
可能原因：无可用副本→预约；达上限→先归还；重复借阅→检查列表；账号异常→联系管理员；有未缴罚款→先缴纳。

**Q: 借阅期限多久？**
普通读者30天（可续2次×15天=最长60天），VIP/教师更长。具体以系统显示为准。

**Q: 如何续借？几次？**
"我的借阅"→点击"续借"按钮。普通读者最多2次，每次延长15天。逾期不可续借。

**Q: 逾期的后果？**
产生罚款（天数×费率×数量），无法续借和新借，影响信用。尽快归还并缴纳罚款。

### 6.3 预约相关问题

**Q: 预约后多久能拿到？**
取决于排队人数和复本数：1人排队且有复本→约1-7天；多人排队→1-4周以上。到馆后需3天内领取。

**Q: 如何取消？**
"我的预约"→仅预约中/待取书状态可取消→点击"取消预约"按钮。频繁取消影响信誉。

**Q: 有效期多长？**
排队期7-14天 + 取书期3天 = 最长约10-17天。过期需重新排队。

### 6.4 座位相关问题

**Q: 如何预约座位？**
座位预约页→选阅览室/日期/时间段→点选绿色座位→确认预约。记得准时签到！

**Q: 迟到多久算违约？**
预约开始后30分钟内需签到（最早可提前15分钟）。超时记违规1次。

**Q: 违约后果？**
累计3次违规→禁止预约7天。被禁止期间其他功能正常。期满自动恢复或申请人工解除。

### 6.5 其他问题

**Q: 推荐浏览器？**
Chrome 90+/Edge 90+/Firefox 88+最佳。需启用JS和Cookie。IE不支持。

**Q: 如何联系管理员？**
服务热线、官方邮箱、现场服务台、在线咨询、微信公众号。准备好问题描述和截图。

**Q: 数据隐私？**
采用加密存储、HTTPS传输、严格权限控制、数据最小化原则。不出售个人信息。

## 7. 错误提示说明

### 7.1 认证相关错误（401）

| 提示信息 | 场景 | 解决方法 |
|---------|------|---------|
| 用户名或密码错误 | 登录 | 检查拼写和大小写，使用"忘记密码" |
| 验证码错误或已过期 | 登录 | 点击刷新重新输入 |
| 账号已被禁用 | 登录 | 联系管理员了解原因并申请解封 |
| Token已过期，请重新登录 | 操作中 | 重新登录，勾选"记住我"可延至7天 |
| 登录失败次数过多，账号临时锁定 | 登录 | 等30分钟解锁或联系管理员 |
| 用户不存在 | 登录/操作 | 确认用户名正确，新用户需先注册 |

### 7.2 权限不足错误（403）

| 提示信息 | 场景 | 解决方法 |
|---------|------|---------|
| 权限不足，无法执行此操作 | 管理操作 | 联系管理员申请权限或使用有权限账号 |
| 您不是管理员，无法访问此页面 | 访问/admin/* | 普通用户使用前台功能 |
| 该资源仅对内部员工开放 | 特定功能 | 确认是否符合条件，申请权限 |
| 您无权操作此借阅记录 | 操作他人记录 | 只能操作自己的记录或到服务台 |

### 7.3 参数校验错误（400）

| 提示信息 | 场景 | 解决方法 |
|---------|------|---------|
| 用户名/密码不能为空 | 注册/登录 | 填写必填字段 |
| 密码长度必须在6-20字符 | 注册/改密 | 调整长度，建议8位以上强密码 |
| 邮箱/手机号格式不正确 | 注册 | 检查格式（user@example.com / 11位纯数字） |
| 验证码不能为空 | 登录 | 输入图片显示的验证码 |
| ISBN格式不正确 | 新增图书 | 使用10或13位标准ISBN |
| 标题/内容长度不符 | 提交咨询 | 标题5-100字，内容10-500字 |
| 请选择分类 | 提交表单 | 从下拉菜单选择 |

### 7.4 服务器错误（500）

| 提示信息 | 场景 | 解决方法 |
|---------|------|---------|
| 服务器内部错误 | 任意操作 | 刷新重试，持续出错联系管理员并截图 |
| 数据库连接失败 | 数据操作 | 稍后重试，持续则数据库有问题 |
| 文件上传失败 | 上传文件 | 检查大小(<10MB)和格式(jpg/png/pdf)，压缩后重试 |
| 导出任务失败 | 数据导出 | 减少数据量后重试 |
| 网关错误(502)/服务暂不可用(503) | 访问系统 | 等待几分钟或查看维护公告 |

### 7.5 业务逻辑错误

**借阅相关（来自BorrowService.java）：**

| 错误提示 | 原因 | 解决方法 |
|---------|------|---------|
| 用户不存在 | 用户名无效 | 确认账号正确 |
| 用户状态异常，无法借阅 | 账号被禁用 | 联系管理员解封 |
| 用户借阅数量已达上限（最大X本） | 已借满 | 先归还一些图书 |
| 图书不存在 | ISBN/ID错误 | 确认图书信息 |
| 图书无可用副本 | 都借出了 | 使用预约功能 |
| 您已借阅该图书，尚未归还 | 重复借阅 | 检查已借阅列表 |

**座位相关（来自SeatService.java）：**

| 错误提示 | 原因 | 解决方法 |
|---------|------|---------|
| 阅览室不存在 | ID错误 | 选择有效的阅览室 |
| 该座位已被预约或使用中 | 选了不可用的座 | 选择绿色(可用)座位 |
| 该座位正在维护中 | 座位故障 | 选择其他座位 |
| 预约冲突（时间段重叠） | 同时段已约 | 更换时间段 |
| 今日预约次数已达上限(3次) | 超过每日限制 | 取消不需要的预约 |
| 违约次数过多，暂时禁止预约 | 累计3次违规 | 等7天恢复或联系管理员 |

**全局异常（来自GlobalExceptionHandler.java）：**
- `UserNotFoundException` → 400 Bad Request + "用户不存在"
- `RuntimeException` → 500 Internal Server Error + 异常消息
