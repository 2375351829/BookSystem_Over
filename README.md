## 📖 项目简介

城市图书馆管理系统是一个面向城市公共图书馆/高校图书馆的现代化、智能化图书管理平台。采用前后端分离架构,支持多角色(读者/管理员/超级管理员)协同工作,提供完整的图书管理、借阅管理、座位预约、用户管理等功能。

### 🎯 设计目标

| 目标       | 说明                              |
| -------- | ------------------------------- |
| **高可用性** | 核心业务7×24小时稳定运行,目标可用性 99.9%      |
| **高性能**  | 接口响应时间 P99 < 500ms,支持并发用户数 500+ |
| **安全性**  | 符合等级保护2.0要求,数据加密传输和存储           |
| **可扩展性** | 模块化设计,支持功能水平扩展和垂直扩展             |
| **易维护性** | 清晰的代码结构、完善的文档、自动化测试覆盖           |

## 🚀 快速开始

### 环境要求

| 环境      | 版本要求    | 说明      |
| ------- | ------- | ------- |
| Java    | JDK 17+ | 后端运行环境  |
| Maven   | 3.6+    | 项目构建工具  |
| Node.js | 16+     | 前端运行环境  |
| MySQL   | 8.0+    | 数据库     |
| Python  | 3.7+    | 启动器(可选) |

### 方式一: 使用启动器 (推荐)

#### 图形界面版

```bash
python launcher_gui.py
```

#### 命令行版

```bash
# 启动所有服务
python launcher.py start

# 查看服务状态
python launcher.py status

# 停止所有服务
python launcher.py stop
```

### 方式二: 手动启动

#### 1. 数据库准备

```sql
CREATE DATABASE CityLibrary CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

修改数据库配置:

```properties
# spring-boot-backend/src/main/resources/application.properties
spring.datasource.url=jdbc:mysql://localhost:3306/CityLibrary
spring.datasource.username=root
spring.datasource.password=your_password
```

#### 2. 启动后端

```bash
cd spring-boot-backend
mvn clean install
mvn spring-boot:run
```

后端服务将在 `http://localhost:8080` 启动

#### 3. 启动前端

```bash
cd vue3-project
npm install
npm run dev
```

前端服务将在 `http://localhost:5173` 启动

#### 4. 访问系统

打开浏览器访问: `http://localhost:5173`

**默认管理员账号:**

| 字段  | 值          |
| --- | ---------- |
| 用户名 | `admin`    |
| 密码  | `admin123` |
